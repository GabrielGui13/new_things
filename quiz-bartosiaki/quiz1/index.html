O Questionário de Estilos de Criatividade Revisado consiste em 8 escalas. Como o questionário usa uma escala de 5 pontos, muitos dos itens devem ser revertidos para pontuação.


I) Medida Global de Capacidade Criativa de Kumar e Holman: a escala consiste em dois itens e mede até que ponto uma pessoa se percebe como criativa. Pontuações mais altas na escala refletem uma maior percepção de criatividade.

II) Crença em processos inconscientes: os itens nesta escala medem até que ponto uma pessoa acredita no processo criativo como perspicaz e inspirador, sobre o qual ela tem pouco controle. Pontuações mais altas na escala indicam maior crença no processo inconsciente como significativo para o trabalho criativo.

III) Uso de técnicas: Os itens desta escala medem até que ponto uma pessoa define estratégias ou técnicas específicas para facilitar seu trabalho criativo. Pontuações mais altas refletem um maior uso de diferentes técnicas.

IV) Uso de outras pessoas: os itens nesta escala medem até que ponto uma pessoa consulta outras pessoas, trabalha com outras pessoas ou forma ideias ou produtos criativos com outras pessoas. Pontuações mais altas indicam maior uso de outras pessoas.
Pontuação:

V) Orientação para o produto final: os itens nesta escala medem até que ponto as pessoas são motivadas a se envolver no trabalho criativo pelo desenvolvimento de um produto final. Pontuações mais altas refletem uma orientação mais alta para o produto.

VI) Controle Ambiental / Autorregulação Comportamental: Os itens desta escala medem até que ponto uma pessoa configura estímulos discriminativos para se autorregular ou facilitar seu trabalho criativo. Pontuações mais altas indicam que a pessoa está estabelecendo um número maior de estímulos discriminativos para facilitar seu trabalho criativo.
Pontuação:

VII) Superstição: Os itens nesta escala medem até que ponto uma pessoa se envolve em comportamento supersticioso para facilitar o trabalho criativo. Pontuações mais altas indicam um comportamento mais supersticioso.
Pontuação:

VIII) Uso dos Sentidos: Os itens desta escala medem até que ponto uma pessoa usa os cinco sentidos para um trabalho criativo. Pontuações mais altas indicam maior uso dos sentidos.
